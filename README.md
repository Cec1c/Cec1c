## 关注方向

我主要做和 AI 辅助工作流、逆向工程、批处理自动化、MCP 集成有关的实用工具。比起“看起来很酷的 demo”，我更喜欢能接到本地环境里、真的减少操作阻力的小工具。

- 接入真实桌面工作流的 AI 工具
- 同时考虑 GUI 和 CLI 的逆向辅助工具
- MCP Server 与本地自动化桥接
- Windows / PowerShell 开发者体验
- Python 快速原型，Go 做轻量原生工具
