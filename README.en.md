## Focus

I build practical developer and automation tools around AI-assisted workflows, reverse engineering, batch processing, and MCP integration.

- AI tooling that plugs into real desktop workflows
- Reverse engineering helpers with GUI and CLI surfaces
- MCP servers and local automation bridges
- Windows/PowerShell developer experience
- Python-first prototypes, Go for compact native utilities
